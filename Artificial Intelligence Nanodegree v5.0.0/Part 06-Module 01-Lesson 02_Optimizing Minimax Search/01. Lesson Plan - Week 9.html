# Lesson Plan: Week 9

The lectures this week address several of the limitations of the basic minimax search algorithm, with an emphasis on techniques that are applicable to the project.

Classroom

  • Watch the Optimizing Minimax Search
  • Complete the "Build an Adversarial Game Playing Agent" project (Remember to start early!)
  • (Optional) Review the Additional Topics lessons, which includes more advanced extensions of Minimax and select readings.

Reading

  • Read AIMA Chapter 5.3; skim AIMA 5.4-5.8
  • Game Tree Searching by Min / Max Approximation by Ron Rivest, MIT
  • (Optional) Deep Blue by IBM
  • (Optional) AlphaGo from DeepMind
