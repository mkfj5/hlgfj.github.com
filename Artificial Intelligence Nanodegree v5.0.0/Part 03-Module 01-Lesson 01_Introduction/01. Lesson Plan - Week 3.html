# Lesson Plan: Week 3

This week you'll focus on classical search algorithms, including depth-first search, breadth-first search, uniform cost search (which is very similar to Dijkstra's algorithm if you're already familiar with that), and A* search. When you're done, you'll have the chance to implement the various search algorithms for the game Pacman to help the titular character find food dots. You can find the exercise here.

# Problem Solving by Search

In this course, we will look into several Search algorithms to solve problems. Problem-solving agents use search algorithms to navigate through a set of state space of the problem and look for the goal states in order to return the solutions if they exist in the state space. We will deep dive into uninformed and informed search strategies in the next lessons to find solutions more efficiently.
