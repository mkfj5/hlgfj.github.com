01. Overview

# Overview

Udacity Workspaces are a development environment in your web browser that can be used to complete projects and classroom exercises without configuring your own local development environment. You cannot create your own Workspaces; they have already been pre-configured in the classroom with all the required data, support libraries, and project files.

All of the projects and classroom exercises in the AIND program have Workspaces available. However, you are not required to use the workspaces—all projects and exercises also include instructions for configuring your own local environment with the required files and libraries. Workspaces can be a convenient option, especially if you have any trouble setting up your own environment.

This lesson is provided as a reference for the Workspaces interface. You do not need to read it all in detail now. Just bookmark the page, and return to it if you choose to use the workspaces and want more information about their capabilities.

The next lesson will explain the steps you would need to complete in order to configure your own development environment. (You can skip the environment setup if you plan to use the workspaces for all projects.)
