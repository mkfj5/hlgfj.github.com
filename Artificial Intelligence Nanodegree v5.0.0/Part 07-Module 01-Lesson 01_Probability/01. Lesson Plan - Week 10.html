# Lesson Plan: Week 10

This week we'll talk about probabilistic models, which will allow your agents to handle uncertainty in the real world with the concept of a "belief state" (a probability distribution over the possible world states). These ideas have been used in robot localization and other real-world tasks, including use in neural networks.

In the Planning lessons, we have learned knowledge-based AI agents that utilize belief state to reason and plan under uncertainty. In this course, we will learn Probabilistic Models and Bayes Networks as the tools for the AI agents to quantify and act under the uncertainty of the surrounding world. Unlike the logical assertions, which see the possible worlds in a set of strict rules, the probabilistic assertions quantify how probable the various worlds are.

We will start this course by refreshing the basic probability terminologies and formulas, including joint probability, conditional probability and Bayes Rule. In the Naive Bayes lesson, we will learn a machine learning algorithm that applies Bayes Rule and assumes that each input variable is independent. In the following Bayes Nets and Inference in Bayes Nets lessons, we will use a special case of graph structure to represent a set of variables and their conditional dependencies, build Bayes Nets with probability distributions, and make inferences from the Bayes Nets based on the queries and evidences. For the final project in this course, we will apply the Hidden Markov Model, a statistical model to compute the transition and unobservable events to estimate the probability of an event.
